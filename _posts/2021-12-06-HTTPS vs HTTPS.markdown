### HTTP vs HTTPS

- **HTTP**

    인터넷 상에서 클라이언트와 서버가 자원을 주고 받을 때 쓰는 통신 규약
    HTTP는 텍스트 교환이므로, 누군가 네트워크에서 신호를 가로채면 내용이 노출되는 보안 이슈가 존재

- **HTTPS**
    
    인터넷 상에서 정보를 암호화하는 SSL 프로토콜을 사용해 클라이언트와 서버가 자원을 주고 받을 때 쓰는 통신 규약
