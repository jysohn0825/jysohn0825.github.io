# Welcome to My Blog! 🎉

안녕하세요! 제 개인 블로그에 오신 것을 환영합니다.

## About This Blog

이 블로그는 **Vue.js**와 **Vite**를 사용하여 구축된 정적 사이트입니다. Jekyll과 같은 기능을 Vue 생태계 안에서 구현했습니다.

### 주요 기능들

- ✅ Vue 3 + Vite 기반
- ✅ Vue Router를 통한 SPA
- ✅ 마크다운 포스트 지원
- ✅ 구문 하이라이팅
- ✅ 반응형 디자인
- ✅ GitHub Pages 배포 지원

## 기술 스택

```javascript
const techStack = {
  frontend: 'Vue 3',
  bundler: 'Vite',
  routing: 'Vue Router',
  markdown: 'Marked',
  highlighting: 'Highlight.js',
  styling: 'CSS3',
  deployment: 'GitHub Pages'
}

console.log('Built with:', techStack)
```
